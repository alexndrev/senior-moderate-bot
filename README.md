# ▹ Installation #
> [!NOTE]  
> For start make sure you have python 3.10+ installed on your machine.
``` Bash
$ git clone https://github.com/alexndrev/senior-moderate-bot
$ cd senior-moderate-bot
$ python3 -m venv .venv
$ .venv/bin/activate
$ pip3 install -r requirements.txt
```

# ▹ Run #
``` Bash
$ python3 main.py
```
